# Transcript: PTT-20260513-WA0080.opus

جی ادر احشیصہ پھر یہ ایک بہم ہے یا وصوصہ ہے اور آمتو پر شتان انسان کے دل میں وصوصہ پیدا کرتا ہے ایبادت کے دوران بھی اور ایبادت کے بعد بھی صرف اُس کی ایبادت کو خراب کرنے کے لیے اُس میں شک پیدا کرنے کے لیے اُس میں دوٹ پیدا کرنے کے لیے شتان ایسرہ کی کوشش کرتا ہے کہ یہ شک آپ کو ہوا یہ دوٹ آپ کو ہوا حلک کروانی کے تکیبن ایک دلنٹے کے بعد تو یہ صرف وصوصہ ہے بہم ہے اور اس کی پیدا کرنے کے لیے اوز آپ یہ کیا کہ انشال آپ کی صیح دروس ہے بیسی آپ کو میں شرن ایک مسر بطارا ہوں کہ اگر آپ کو یہ شک وہی پے ہو جاتا جب آپ صیح کر رہے تھے اور صفابی تھے تو آپ کو گرشک تھا شک ہو جاتا گر آپ کو کہ آپ کے چکر چکر چی ہو مئی یہ آٹ مئی ہے تو پھر آپ کو وہاں پر ایک چکر اہجیاتا لگالینا چاہیے تھا اور مربا پہ اپنی صحیح ختم کرتے کیوں کہ صفاب مربا کی صحیح کے جو صحیح کے جو صحیح کے جو صحیح کر مربا بھی ختم کرتے وہ صفابی ختم نہیں ہوتے تو آپ اس ایک چکر اور لگال دیتے اور آپ کی صحیح گوہ دروسیتی تو اس لیے اوز چوگی اشک آپ کو بعد میں ہوا ہے تو اس کی پروان کریں تو آپ کو یہ سمس لنی ہے
