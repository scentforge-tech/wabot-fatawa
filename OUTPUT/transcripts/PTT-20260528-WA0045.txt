# Transcript: PTT-20260528-WA0045.opus

اسم ایک محمت اللہ ورکہ دوساجت خان دائیں جب آپ گیارا بجے اپنی ہوتے پہنے گیت ہے تو اس کے بعد آپ دو تین گھنٹے چار گھنٹے آپ رس کر کے فرش ہوگ کیا بجاز سکتے ہرم میں کیوں کہ آپ کو یہ پتا تھا کہ رااد مینام مگدار نواجب ہے پیلم تھا آپ کو رااد مینام مگدار نواجب ہے جیکن اس کی بابجود آپ رااد کو گیت وافی فاضہ کرنے اور اسی طرح سای کرنے کے لیراد کو کیوں گی آپ جب آپ کی اپتا تھا کہ رااد مینام مگدار نی ہے تو آپ کو یہ بھی بتا ہوگا کہ توافی فاضہ اور سای جہے کو دوسطلحج کو کرنا چھروری نہیں ہے تو جو کام ضروری تھا وہ آپ نے چھور دیا جو کام ضروری نہیں تھا کل دوسطلحج کو وہ آپ کرنے چھلے گے تو اس کے خمیادہ بگتو بھی آپ میں کیا کر سکتا ہے اس میں آپ کی قفلت شامل ہے کہ دن کے بقھ جا آپ جا سکتے تھے اور بہاں سے جلدی فارہ ہو کے باپا سا سکتے تھے ایک یہ آبشان تھا بھکی پاس دوسطلحج انیے تھا کہ آپ کل یہ توافی فاضہ نہ کرتے سای نا کرتے کیوں کہ آپ کہیں کہ آپ تکاوٹ بہت جاتتے تو بہی تکاوٹ تھی تو آپ ایدری رہتے آپ اپنے ہٹل پیر رہتے اور شامل کو آپ چلے جاتے منا کی اندر رات گذارتے یہ واجپورا کرتے اور توافی فاضہ اور سای عاج کر سکتے تھے کلکہ سکتے تھے یہ آپ کی پلینگی غلت آپ کی ترطیبی غلت اب اس کے خمیادہ بگتو خودی
